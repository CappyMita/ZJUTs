股票市场的板块轮动分析（时间序列多元扩展）

课程对应点： 多元正态分布检验、均值向量检验、主成分分析的时序应用。

背景： 选取A股市场5个不同行业板块（如：银行、消费、医药、科技、能源）的日收益率数据（共约250个交易日）。

编程任务：

多元正态性检验： 对5个板块的日收益率向量进行多元正态性检验（如Mardia检验或Q-Q图）。通常会发现厚尾现象，讨论其金融含义。//唐思承

滚动窗口PCA： 设定窗口大小=60天，步长=1天。在每个窗口内对5个变量做PCA，提取第一主成分的贡献率。画出贡献率的时间序列图，观察当市场出现系统性风险（如熊市）时，第一主成分贡献率是否显著上升（意味着同涨同跌）。//郭庆

均值向量检验： 将数据分为“上涨市”和“下跌市”两个阶段（以大盘指数为基准），使用Hotelling T²检验判断两个阶段下5个板块的均值向量是否发生了显著偏移（验证板块轮动是否存在）。//刘孔嘉

可视化： 画出5个板块收益率的平行坐标图，展示牛市与熊市的不同路径。

# ================= 1. Mardia 多元正态性检验 =================
def manual_mardia_test(X):
    n, p = X.shape
    mean_vec = np.mean(X, axis=0)
    cov_matrix = np.cov(X, rowvar=False, ddof=0)  # 总体协方差 (MLE)

    inv_cov = np.linalg.inv(cov_matrix)
    diff = X - mean_vec

    # 马氏距离平方: D_i^2
    mahal_dist_sq = np.sum(np.dot(diff, inv_cov) * diff, axis=1)

    # 内积矩阵: inner_prod[i,j] = (x_i-μ)' Σ⁻¹ (x_j-μ)
    inner_prod = np.dot(np.dot(diff, inv_cov), diff.T)

    # Mardia 偏度 b_{1,p} = (1/n²) ΣᵢΣⱼ [inner_prod]³ ——
    b1_p = np.sum(inner_prod ** 3) / (n ** 2)

    # 峰度统计量
    b2_p = np.mean(mahal_dist_sq ** 2)
    expected_b2_p = p * (p + 2)

    # 偏度检验自由度 = p(p+1)(p+2)/6 ——
    k_squared = n * b1_p / 6.0
    df_skew = p * (p + 1) * (p + 2) / 6.0
    p_value_skew = 1 - chi2.cdf(k_squared, df_skew)

    # 峰度检验
    std_b2_p = np.sqrt(8 * p * (p + 2) / n)
    z_score = (b2_p - expected_b2_p) / std_b2_p
    p_value_kurt = 2 * (1 - norm.cdf(np.abs(z_score)))

    return {
        "Skewness (b1,p)": b1_p,
        "Kurtosis (b2,p)": b2_p,
        "Expected Kurtosis": expected_b2_p,
        "Skewness Chi2 Stat": k_squared,
        "Skewness df": df_skew,
        "Skewness P-value": p_value_skew,
        "Kurtosis Z Stat": z_score,
        "Kurtosis P-value": p_value_kurt,
    }


# ================= 2. PCA (基于特征值分解) =================
def manual_pca(X):
    mean = np.mean(X, axis=0)
    std = np.std(X, axis=0, ddof=1)
    X_scaled = (X - mean) / std
    cov_matrix = np.cov(X_scaled, rowvar=False)
    eigenvalues, _ = np.linalg.eigh(cov_matrix)
    sorted_eigenvalues = np.sort(eigenvalues)[::-1]
    pc1_variance = sorted_eigenvalues[0] / np.sum(sorted_eigenvalues)
    return pc1_variance


# ================= 3. Hotelling T² 检验 =================
def manual_hotelling_t2(X1, X2):
    n1, p = X1.shape
    n2, p = X2.shape
    mean_diff = np.mean(X1, axis=0) - np.mean(X2, axis=0)

    cov1 = np.cov(X1, rowvar=False, ddof=1)
    cov2 = np.cov(X2, rowvar=False, ddof=1)
    pooled_cov = ((n1 - 1) * cov1 + (n2 - 1) * cov2) / (n1 + n2 - 2)

    inv_pooled_cov = np.linalg.inv(pooled_cov)
    mahalanobis_sq = np.dot(np.dot(mean_diff.T, inv_pooled_cov), mean_diff)
    
    t_squared = (n1 * n2 / (n1 + n2)) * mahalanobis_sq

    denominator_df = n1 + n2 - p - 1
    f_stat = (denominator_df / ((n1 + n2 - 2) * p)) * t_squared
    p_value = 1 - f_dist.cdf(f_stat, p, denominator_df)
    return t_squared, p_value
