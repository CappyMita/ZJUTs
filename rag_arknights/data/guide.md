# 《明日方舟》集成战略“岁的界园志异”综合攻略

## 一、 模式简介与核心机制

“岁的界园志异”（界园肉鸽）采用独特的双层结构设计。一楼为常规探索节点，玩家需通过“梦境节点”进入二楼“秃洞”探险。秃洞是一张平面图，极大地丰富了界园的可探索度。由于秃洞的收益极高（大多数票券来源、特定遗物及多结局触发均限定于此），而常规节点收益偏低，玩家在实际游玩中通常需要半强制性地优先探索秃洞。

### 1. 核心机制解析
- **岁时**：岁兽代理人遗留在此的权能回响。在进入特定层时触发，岁时效果会叠加存在且无法消除。游玩时难度越高，岁时效果越强大，部分Debuff极其恶心且无法主动消除。
- **通宝**：结合了随机性与策略性的机制。玩家掷出的通宝数量小于钱盒中的总数，无法完全随心所欲。可以通过获取强力通宝替换旧通宝，或消耗票券重投来提高触发概率。本质上属于概率触发的收藏品。
- **饲烛客**：干员成为饲烛客后，主要作用是在“岁兽残识”中作战。成为饲烛客的主要途径是在“指点迷津”节点选择使用留存招募券进行特殊招募（可减免2点希望并一次性使多名干员成为饲烛客）。需注意，饲烛客机制会使所有晋升券、古旧铸物等晋升藏品收益大幅降低。
- **前瞻性投资系统**：局内类似银行的系统，可存入或提取源石锭（有上限且提取消耗递增）。余额在每次探索中继承，累积余额可解锁投资奖赏（如商店额外出售商品、解锁特定藏品等）。

### 2. 特勤干员“电弧”养成与使用
本次赠送干员“电弧”对游戏体验影响极大，其养成完全依赖于界园肉鸽内的游玩与特定培训任务，无法使用常规材料培养。携带电弧可为探索提供额外资源（如多留存1张招募券、护盾+1）及事件额外选项。

- **战斗精通1**：每到达新的一层，电弧依次获得效果：召唤物部署费用-9、可同时部署数量+2、攻击速度+35、不再消耗部署位。建议至少拿到前两个效果，大幅减少费用压力并提升输出能力。
- **后备精通**：电弧及其召唤物生命值、防御力+30%，法术抗性+15。若挑战高难（如N15），建议获取。
- **模组升级**：解锁“电弧特勤证章”后，召唤物持有上限+5，且召唤物被击倒或撤退时有25%概率被回收，有效补充技能CD期间的召唤物数量。

## 二、 各结局达成条件与流程

### 二结局
1. 在“岁兽残识”的“抉择”节点中触发事件【榆蝶】。
2. 消耗30源石锭直接获得藏品【隐蝶墨】，或在事件中选择“靠运气”获取。
3. 获得【隐蝶墨】后，在战斗关卡中出现“颉的龙爪”，将其生命值削减至一半以下，获得藏品【墨化残碑】。
4. 在第五层的“命运所指”节点使用【墨化残碑】，即可进入二结局BOSS战。

### 三结局
**注意：最终关卡仅允许“饲烛客”角色参与挑战！**
1. 在“岁兽残识”的“杂疑”节点触发事件【执棋】，需队伍中包含五名“饲烛客”干员。
2. 选择“掀翻棋盘”获得藏品【忘生珍珑】。
3. 在“指点迷津”节点选择“棋盘”分支，完成特殊战斗后获得藏品【云与漆】。
4. 击败第五层BOSS后，自动进入第六层解锁三结局。（BOSS为“臭棋篓子”，需注意避免干员被棋子围困）

### 四结局
**注意：最终关卡仅允许“入卷”角色参与挑战！**
1. 在“岁兽残识”的“易与”节点，消耗5点目标生命值购买藏品【无封长盒】。
2. 持有【无封长盒】时，在“先行一步”节点选择“提请起盒”令干员入卷，获得藏品【无墨长卷】（希望+3，使险路恶敌更艰难）。
3. 击败第五层BOSS“易”或“岁”后进入第六层。
4. 通过“命运所指——大狩维天”节点（此后仅能使用“饲烛客”），最终击败BOSS“后兽”与“矩兽”解锁四结局。

### 五结局
1. **前置条件**：需先打过一次任意结局。
2. 在“岁兽残识”的“故肆”节点，选择“点墨生器”，消耗5点目标生命值上限获得藏品【追忆仪】（失去所有源石锭和希望，后续战斗经验和源石锭减半）。建议在第四层获取，第五层通过“误入奇境”过前置关卡。
3. 拿到【追忆仪】后，进入“误入奇境”节点选择“凝视仪器”，进入前置关卡“录忆”，通关获得藏品【小磨唧】（费用回复速度减半）。
4. 通关第五层或第六层后，进入特殊层通关五结局“定乾坤”。

## 三、 推荐干员与藏品选择

由于整体难度提高、鬼图较多且一楼收益低，对干员精一即可用的要求较高。推荐阵容如下：

- **先锋**：伊内丝（输出回费两开花）。
- **近卫**：惊蛰（近卫唯一推荐）。
- **重装**：黍或余（炎国户口，重装券随便抓）。
- **特种**：新能（核心）、夜刀（应对各种鬼图）、地刺（大炎压缩包）。
- **狙击**：EW（精二的神，精一路边一条）、蕾缪安（二三结局可用，搭配弹药类藏品）。
- **术士**：香蕉姐或异拉澄闪（前者搭配掌机，后者打全图）。
- **医疗**：凯尔希（老猫优于小猫）、艾雅法拉（奶全图含金量高）。

**藏品倾向**：大量弹药类和召唤物类藏品较为强势。注意避开收益极低的“化境格子”和随机性过大的铜钱增益。

## 四、 难度选择与资源管理技巧

### 难度推荐
- **N7**：推荐用于熟悉二结局技能。N7与N9均可吃到“启工记”收益（商店额外出售两个藏品），抢商店收益极高，且N7可一步到位完成电弧专二和专三的技能要求。
- **N9**：能吃到“设园通鉴”希望+1的收益。配合开局概率刷出的“烧水壶”（一希望一生命），可实现任意分队主C加电弧开局，借助战抢商店。

### 资源管理
- **希望与招募券**：前期需合理规划，特别是在“指点迷津”节点利用留存招募券进行特殊招募以获取“饲烛客”身份。
- **源石锭**：合理利用“前瞻性投资系统”进行存取，为后续探索积累优势。
- **科技树**：通过累积200积分获取“失落智识”来点亮科技树，获取局内增益（如“电表倒转”等逃课向增益优先点亮）。
- **狭路相逢节点**：在界园中该节点收益弱化且极其恶心（高台打隐匿等），若未解锁请勿轻易解锁。
